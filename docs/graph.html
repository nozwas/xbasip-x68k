xbasip: X-BASIC like package for micropython-x68k

xbasipパッケージ

graphモジュール

  • pset(x, y, color)

    • ドットを描画します。
    • x -- X座標
    • y -- Y座標
    • color -- カラーコード
      • 16色モード、256色モードの場合はパレットコード
      • 65536色モードの場合は64K色コード（0b_GGGGG_RRRRR_BBBBB_L：RGB各5ビット、L輝度1ビット）

  • line(x1, y1, x2, y2, color, style=0xffff)

    • 直線を描画します。
    • x1 -- 始点X座標
    • y1 -- 始点Y座標
    • x2 -- 終点X座標
    • y2 -- 終点Y座標
    • color -- カラーコード
    • style -- ラインスタイル。16ビットのパターンと見なし、実線（0xffff）や点線（0xaaaa、0xcccc）などを指定することができます。

  • line_to(x, y, color, style=0xffff)

    • 前回のline終点またはpset点を始点に直線を描きます。

  • box(x1, y1, x2, y2, color, style=0xffff)

    • 四角形を描画します。

  • fill(x1, y1, x2, y2, color)

    • 中を塗りつぶした四角形を描画します。

  • circle(x, y, r, color=None, start=0, end=360, ratio=256)

    • 円や楕円、扇形、円弧を描画します。
    • x -- 中心点X座標
    • y -- 中心点Y座標
    • r -- 半径
    • color -- カラーコード
    • start -- 開始角度（度）
    • end -- 終了角度（度）
    • ratio -- 扁平率
      • ratio＜256 : 横長楕円（rを横半径とする）
      • ratio＝256 : 真円
      • ratio＞256 : 縦長楕円（rを縦半径とする）
    • 開始角度や終了角度に負数を与えると扇形を描くことができます。

  • paint(x, y, color, buf=None)

    • 単色領域内を塗りつぶします。
    • x -- 塗りつぶし開始点X座標
    • y -- 塗りつぶし開始点Y座標
    • color -- カラーコード
    • buf -- 作業用バッファ（bytearray）。省略時は内部で1024バイト確保します

  • get(x1, y1, x2, y2, buf=None)

    • グラフィック画面から矩形領域をデータとして取り込みます。
    • x1 -- 始点X座標
    • y1 -- 始点Y座標
    • x2 -- 終点X座標
    • y2 -- 終点Y座標
    • buf -- データバッファ（bytearray）。省略時は内部で確保します
    • 戻り値 -- データバッファを返します

  • put(x1, y1, x2, y2, buf)

    • データを矩形領域に描画します。
    • x1 -- 始点X座標
    • y1 -- 始点Y座標
    • x2 -- 終点X座標
    • y2 -- 終点Y座標
    • buf -- データバッファ（bytearrayもしくはbytes）

  • get2(x1, y1, x2, y2)

    • グラフィック画面から矩形領域をデータとして取り込みます。データ構造はget()、put()と少し異なります。データの先頭に幅、高さ、色モードの3ワードが付与されます。
    • x1 -- 始点X座標
    • y1 -- 始点Y座標
    • x2 -- 終点X座標
    • y2 -- 終点Y座標
    • 戻り値 -- データバッファ（bytearray）を返します。データの先頭に下記の3ワードが付与されます
      • 矩形領域の幅（16bit） : 画素数
      • 矩形領域の高さ（16bit） : 画素数
      • 色モード（16bit） : 0x000f（16色）、0x00ff（256色）、0xffff（65536色）
    • window()によるクリッピングエリアの設定はこの関数に反映されません。

  • put2(x, y, buf, mask=None)

    • 矩形領域データとして取り込みます。データ構造はget()、put()と少し異なります。データの先頭に幅、高さ、色モードの3ワードが必要です。また、マスク色を指定して透過描画することができます。
    • x -- 始点X座標
    • y -- 始点Y座標
    • mask -- マスク色カラーコード。データ中のこの色の部分は描画しません。描画先が透過します
    • buf -- データバッファ（bytearrayもしくはbytes）。データの先頭に下記の3ワードが必要です
      • 矩形領域の幅（16bit） : 画素数
      • 矩形領域の高さ（16bit） : 画素数
      • 色モード（16bit） : 0x000f（16色）、0x00ff（256色）、0xffff（65536色）
    • window()によるクリッピングエリアの設定はこの関数に反映されません。

  • symbol(x, y, str, xmag, ymag, font_size, color, dir=0)

    • グラフィック画面に文字列を描画します。
    • x -- 始点X座標
    • y -- 始点Y座標
    • str -- 文字列
    • xmag -- 横方向の倍率（1～255）
    • ymag -- 縦方向の倍率（1～255）
    • font_size -- フォントサイズ
      • 0 : 6×12、12×12
      • 1 : 8×16、16×16
      • 2 : 12×24、24×24
    • color -- カラーコード
    • dir -- 方向
      • 0 : 0度
      • 1 : 90度
      • 2 : 180度
      • 3 : 270度

  • point(x, y)

    • 指定位置のドットのカラーコードを返します。
    • x -- X座標
    • y -- Y座標
    • 戻り値 -- カラーコード

  • palet(pal, color)

    • グラフィックのパレットを設定します。
    • pal -- パレット番号
    • color -- カラーコード
      • 0～65535 : 64K色コード（0b_GGGGG_RRRRR_BBBBB_L：RGB各5ビット、L輝度1ビット）

  • rgb(r, g=None, b=None)

    • R、G、Bの成分を0～31で指定し64K色コードを求めます。最下位の輝度1ビットは0固定です。
    • r -- R成分（0～31）
    • g -- G成分（0～31）
    • b -- B成分（0～31）
    • 戻り値 -- 64K色コード（0b_GGGGG_RRRRR_BBBBB_0：RGB各5ビット、L輝度1ビット＝0）
    • 引数を１つのtupleで与えることも可能です。

  • rgb24(r, g=None, b=None) -- RGB各8bit、tupleで与えることもできます

    • R、G、Bの成分を8ビットで指定し64K色コードを求めます。最下位の輝度1ビットは0固定です。
    • r -- R成分（0～255）
    • g -- G成分（0～255）
    • b -- B成分（0～255）
    • 戻り値 -- 64K色コード（0b_GGGGG_RRRRR_BBBBB_0：RGB各5ビット、L輝度1ビット＝0）
    • 引数を１つのtupleで与えることも可能です。

  • hsv(h, s, v)

    • H、S、Vの成分から64K色コードを求めます。最下位の輝度1ビットは0固定です。
    • h -- 色相（H）成分（0～191）
    • s -- 飽和度（S）成分（0～31）
    • v -- 明るさ（V）成分（0～31）
    • 戻り値 -- 64K色コード（0b_GGGGG_RRRRR_BBBBB_0：RGB各5ビット、L輝度1ビット＝0）
    • 引数を１つのtupleで与えることも可能です。

  • contrast(level)

    • 画面全体のコントラストを指定します。
    • level -- コントラストレベル
      • 0～15 : コントラスト低い～高い

  • window(x1, y1, x2, y2)

    • クリッピングエリアを設定します。
    • x1 -- クリッピングエリアの左上X座標
    • y1 -- クリッピングエリアの左上Y座標
    • x2 -- クリッピングエリアの右下X座標
    • y2 -- クリッピングエリアの右下Y座標

  • wipe() -- グラフィック画面消去

    • アクティブページのクリッピングエリア内を消去します。

  • apage(page)

    • アクティブページ（描画ページ）を設定します。
    • page -- ページ番号（0～3）

  • vpage(page_bit) -- 表示ページ

    • 各グラフィックページのON/OFFを設定します。
    • page_bit -- 各ページに対応するビットを1にすると表示ON、0にすると表示OFF
      • bit 3 : ページ3
      • bit 2 : ページ2
      • bit 1 : ページ1
      • bit 0 : ページ0

  • home(page, x, y)

    • 表示画面の左上に対応する実画面の座標を設定します。
    • page -- ページ番号（0～3）
    • x -- 表示画面の左上に対応する実画面のX座標
    • y -- 表示画面の左上に対応する実画面のY座標

  • scroll(page, x, y)

    • グラフィック、テキストの表示座標を設定します。
    • page
      • 0～3 : グラフィックページ番号
      • 8 : テキスト
    • x -- 表示画面の左上に対応する実画面のX座標
    • y -- 表示画面の左上に対応する実画面のY座標
