xbasipパッケージ

musicモジュール

  • m_init(mode=0)

    • FM音源を初期化します。
    • mode -- 初期化モード
      • 0 : トラックバッファの初期化
      • 1 : トラックバッファ、音色データの初期化

  • m_alloc(trk, size)

    • トラックバッファを確保します。
    • trk -- トラック番号（1～80）
    • size -- バッファサイズ（バイト）
    • 全トラックバッファ内のデータはクリアされます。

  • m_free(trk)

    • トラックバッファの空き容量を返します。
    • trk -- トラック番号（1～80）
    • 戻り値 -- 空き容量（バイト）

  • m_assign(ch, trk)

    • チャネルにトラックバッファを割り当てます。
    • ch -- チャネル番号（1～25）
    • trk -- トラック番号（1～80）

  • m_play(*ch_s)

    • 演奏を開始します。
    • *ch_s -- チャネル番号（1～25）。ただしzmusic2.x常駐時はトラック番号（1～80）を意味します
    • 任意個数の引数を与えられます。引数を省略するとすべてのチャネルの演奏を開始します。

  • m_stop(*ch_s)

    • 演奏を停止します。
    • *ch_s -- チャネル番号（1～25）。ただしzmusic2.x常駐時はトラック番号（1～80）を意味します
    • 任意個数の引数を与えられます。引数を省略するとすべてのチャネルの演奏を停止します。

  • m_cont(*ch_s)

    • 演奏を再開します。
    • *ch_s -- チャネル番号（1～25）。ただしzmusic2.x常駐時はトラック番号（1～80）を意味します
    • 任意個数の引数を与えられます。引数を省略するとすべてのチャネルの演奏を再開します。

  • m_stat(ch=0)

    • 演奏状態を返します。
    • ch -- チャネル番号（1～25）
    • 戻り値
      • 0 : 停止中
      • 1 : 演奏中
    • 引数を省略した場合bit 0～bit 24にチャネル1～25の状態を返します。

  • m_tempo(tempo)

    • テンポを設定します。
    • tempo -- 1分間に4分音符を演奏する回数（20～300）。初期状態は120

  • m_trk(trk, mml)

    • MMLデータをトラックバッファに書き込みます。
    • trk -- トラック番号（1～80）
    • mml -- MMLデータ（bytearrayもしくはbytes）

  • m_atoi(trk)

    • トラックバッファのアドレスを取得します。
    • trk -- トラック番号（1～80）
    • 戻り値 -- トラックバッファの先頭アドレス

  • m_vset(vo, buf)

    • 音色データを設定します。
    • vo -- 音色番号（1～200）
    • buf -- 音色データ（bytesarrayもしくはbytes）。55バイト

  • m_vget(vo, buf=None)

    • 音色データを取り出します。
    • vo -- 音色番号（1～200）
    • buf -- 取り出し用のデータバッファ（bytesarray）。55バイト。省略時は内部で確保して返します
    • 戻り値 -- 音色データ（bytearray）。55バイト

X68000メモ

musicモジュールを使用するには、opmdrv3.xもしくはzmusic2.xをシステムに組み込んでおく必要があります。ただし、opmdrv3.xとzmusic2.xにはいくつかの仕様の違いがあり、本ライブラリをする場合の動作にも差異が生じることがあります。例えば、opmdrv3.xでのm_play()はX-BASIC同様にチャネルを指定して演奏しますが、zmusic2.xはトラックを指定して演奏します。多くの場合チャネルとトラックは１対１の関係で使われますが、そうではない場合に動作が異なります。m_stop()、m_cont()も同様です。また、opmdrv3.xでは音色番号1～68に対し下記のプリセットデータが組み込まれていますが、zmusic2.xには組み込まれていません。
